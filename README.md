# Local Development 

## Contract
1. clone the repo
2. cd into ./contracts/
3. nvm use
4. npm install
5. npm run dev
6. visit http://localhost:8080

## Client
1. clone the repo
2. cd into ./client/
3. nvm use
4. npm install
5. npm run dev
6. visit http://localhost:8081
